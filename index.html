slinkydraw

by christine cha

SPACE ..... start / stop drawing

S ......... save the image

Q ......... increase radius value

R ......... increase red value

G ......... increase green value

B ......... increase blue value

A ......... increase opacity value

+ SHIFT ... decrease value


C ......... clear image

Z ......... invert background color
